# Records Table Component

A simple React table component for managing records with CRUD operations, search, and form validation. Built with TypeScript and Ant Design.

## Features

- Create, read, update, and delete records
- Search by each column
- Form validation for name, date, and value
- Responsive table with action buttons

## Setup

Install dependencies:

```bash
npm install
```

## Scripts

- `npm run dev` — Start development server
- `npm run build` — Build for production
- `npm run preview` — Preview production build
- `npm run deploy` — Deploy app

## Usage

**Add a record** — Click "Add row" button, fill the form, and submit

**Search** — Type in the search field to filter records by name

**Edit** — Click the edit icon in the Actions column to modify a record

**Delete** — Click the delete icon to remove a record

## Tech Stack

- React 19
- TypeScript
- Ant Design
- Vite
